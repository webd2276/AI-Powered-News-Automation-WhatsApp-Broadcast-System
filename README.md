# AI-Powered-News-Automation-WhatsApp-Broadcast-System
An advanced no-code/low-code automation pipeline built in n8n that aggregates live news from multiple RSS sources, processes it using AI, and delivers concise, formatted summaries directly to WhatsApp in real time.
